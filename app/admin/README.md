# Admin Dashboard Documentation

## Overview

The SCOAN Admin Dashboard is a comprehensive content management system designed for non-technical users to manage website content, media, users, and settings through an intuitive interface.

## Features

### 1. Content Management
- **Pages**: Create, edit, and manage static pages with WYSIWYG editor
- **Sermons**: Upload and manage sermon videos with thumbnails and metadata
- **Media Library**: Centralized file management for images, videos, and documents

### 2. User Management
- **Role-Based Access Control**: Admin, Editor, and Contributor roles
- **User Permissions**: Granular access control based on user roles
- **Activity Logging**: Track user actions and changes

### 3. System Features
- **Auto-Save**: Automatic saving of content every 30 seconds
- **Version Control**: Track changes and restore previous versions
- **Preview Functionality**: Preview changes before publishing
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## User Roles and Permissions

### Admin
- Full access to all features
- User management
- System settings
- Content moderation
- Analytics and reports

### Editor
- Create and edit content
- Media management
- Page management
- Sermon uploads
- Gallery management

### Contributor
- Create content (pending review)
- Upload media files
- Edit own content
- View-only access to others
- Limited settings access

### Login

2. Use the following demo credentials:

   - **Editor**: editor@example.com / password123
   - **Contributor**: contributor@example.com / password123

### Dashboard Navigation
The sidebar provides access to all major sections:
- **Dashboard**: Overview and quick stats
- **Pages**: Manage website pages
- **Sermons**: Upload and manage sermon content
- **Media Library**: File management
- **Users**: User account management
- **Settings**: System configuration

## Content Management Guide

### Creating/Editing Pages
1. Navigate to **Pages** section
2. Click **Add New Page** or select existing page
3. Use the WYSIWYG editor to format content
4. Add SEO metadata (title, description, keywords)
5. Upload featured image
6. Set page status (Draft, Published, Archived)
7. Click **Save** or **Publish**

### Uploading Sermons
1. Navigate to **Sermons** section
2. Click **Upload Sermon**
3. Fill in sermon details:
   - Title and description
   - Preacher name
   - Date and duration
   - Category and tags
4. Upload video file
5. Upload thumbnail image
6. Click **Save**

### Managing Media
1. Navigate to **Media Library**
2. Click **Upload Files**
3. Select files to upload (images, videos, documents)
4. Files are automatically organized by type
5. Use search and filters to find specific files
6. Download, preview, or delete files as needed

## User Management

### Adding Users
1. Navigate to **Users** section
2. Click **Add User**
3. Enter user details:
   - Name and email
   - Role (Admin, Editor, Contributor)
   - Initial password
4. Set user status (Active/Inactive)
5. Click **Create User**

### Managing User Permissions
- Use the **Users** section to view all users
- Click **Edit** to modify user details
- Use role dropdown to change user role
- Toggle status switch to enable/disable access
- Click **Reset Password** to send password reset email

## Settings Configuration

### General Settings
- **Site Name**: Website title
- **Site Description**: Meta description
- **Site URL**: Website URL
- **Admin Email**: Contact email address

### User Management Settings
- **Enable Registration**: Allow new user registration
- **Require Approval**: New users require admin approval
- **Maintenance Mode**: Temporarily disable site access

### Notification Settings
- **Email Notifications**: Enable/disable email alerts
- **Auto Backup**: Automatic data backup
- **Backup Frequency**: Daily, weekly, or monthly

## Best Practices

### Content Management
1. **Use Descriptive Titles**: Clear, concise page titles
2. **Optimize Images**: Compress images before uploading
3. **Add Alt Text**: Include alt text for accessibility
4. **Regular Updates**: Keep content fresh and relevant
5. **SEO Optimization**: Use relevant keywords in content

### User Management
1. **Strong Passwords**: Enforce strong password policies
2. **Regular Reviews**: Periodically review user access
3. **Role Assignment**: Assign minimum required roles
4. **Activity Monitoring**: Check activity logs regularly

### Media Management
1. **File Organization**: Use descriptive file names
2. **Image Optimization**: Resize images for web use
3. **Copyright Compliance**: Ensure proper licensing
4. **Regular Cleanup**: Remove unused files periodically

## Troubleshooting

### Common Issues

**Login Problems**
- Check credentials
- Clear browser cache
- Contact admin for password reset

**Upload Issues**
- Check file size limits
- Verify file formats
- Check internet connection

**Editor Issues**
- Refresh browser
- Clear browser cache
- Try different browser

**Permission Errors**
- Check user role
- Contact admin for access
- Verify login status
